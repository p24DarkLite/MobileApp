# Smart Shopping List 

Мобільний застосунок на **React Native (Expo)** для зручного створення списків покупок, контролю бюджету та відстеження витрат.

## Особливості
* **Керування списками**: Створюйте списки за категоріями (Їжа, Одяг, Техніка тощо).
* **Контроль бюджету**: Встановлюйте ліміт для кожного списку та бачте витрати в реальному часі.
* **Мультивалютність**: Підтримка UAH (грн), USD ($) та EUR (€).
* **Мультимовність**: Повна локалізація інтерфейсу (Українська / Англійська).
* **Темна тема**: Адаптивний дизайн для комфортного використання.

## 🛠 Технологічний стек
* **Framework**: [Expo](https://expo.dev/) (React Native)
* **State Management**: [Zustand](https://github.com/pmndrs/zustand) (для глобального стану `useStore`)
* **Icons**: `@expo/vector-icons` (Ionicons)
* **Navigation**: Expo Router / React Navigation
